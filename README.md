# dena-review

## 四目並べ
二人のプレイヤーが縦6横7のボードの上で交互にコマを下から積み重ねて、先に縦、横、斜めいずれかに直線状に4つ並べた方が勝つゲーム。
誰も並べられなかった場合、引き分けとなる。
